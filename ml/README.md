## Начало работы
0) установить uv
1) `cd ml`
2) `uv sync`
3) для входа в venv команда `.venv\Scripts\activate` или выбрать в vscode при запуске

## Работа с образом
1) Cборка образа `docker build -t ml .`
2) Запуск образа `docker run -it --rm ml /bin/bash`
